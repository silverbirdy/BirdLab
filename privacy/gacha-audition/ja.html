Gacha! オーディション — プライバシーポリシー

アプリ名: Gacha! オーディション（アイドルカードライブ）
パッケージ名: net.birdlab.gachaaudition
施行日: 2026年5月28日
最終更新日: 2026年5月28日

Gacha! オーディション（以下「本アプリ」）は、利用者の個人情報を大切に取り扱い、関連法令を遵守して利用者の権利保護に努めます。本ポリシーは、本アプリが収集する情報とその利用方法を説明します。

1. 収集する情報

本アプリは、氏名・メールアドレス・電話番号・住所などの個人を特定できる情報を一切収集しません。

ただし、広告表示のため Google AdMob SDK を通じて以下の情報が収集される場合があります:

  • 広告 ID (AAID) — Google が提供するリセット可能な広告識別子。パーソナライズ広告の表示目的。
  • 端末識別子 — 端末モデル、OS バージョン、言語設定。
  • おおよその位置情報 — IP アドレスから推定する広告地域化のため。

これらの情報は Google AdMob によって収集・処理され、本アプリ開発者のサーバーには送信されません。

2. AI 歌詞生成時に送信される情報

「作曲」機能を使用する際、楽曲生成に必要な以下の非識別情報が、本アプリ開発者が運用する Cloudflare Worker（プロキシ）を経由して Google Gemini API に送信されます:

  • 選択したジャンル（例: cute, cool, ballad など）
  • 編成カードのゲーム内ニックネーム（アプリが提供する架空のアイドル名）
  • クオリティスコア（整数）

これらに個人を特定できる情報は含まれず、歌詞生成の応答返却後ただちに破棄されます。本アプリはこれを永続的に保存したり分析に使用したりしません。Gemini API のデータ取り扱いについては Google のポリシーに従います。

3. 収集しない情報

  • 氏名・メール・電話番号・住所などの個人識別情報
  • 正確な GPS 位置
  • 写真、動画、連絡先、カレンダー
  • 音声、健康、金融情報
  • ユーザーアカウント情報（本アプリにはアカウント機能がありません）

4. 端末内に保存される情報

利用者がアプリ内で生成する以下の情報は端末内にのみ保存され、開発者サーバーには送信されません:

  • ガチャ結果および保有カード一覧
  • 編成したユニット情報
  • 制作した楽曲、歌詞、リリース・販売履歴
  • ライブスコア、ファン数、ランキング進行状況
  • ゲーム内通貨残高（コイン、チケットなど）
  • 設定（言語、サウンドなど）

これらは、アプリの削除またはアプリ内のデータリセット機能で永久に削除されます。

5. 収集目的

  • Google AdMob による広告収益化
  • AI 歌詞生成機能の提供（Gemini API 呼び出し）
  • アプリの安定性・パフォーマンス改善（Google Play 標準診断機能）

6. 第三者提供

本アプリは収集した情報を第三者に販売しません。ただし広告表示および AI 機能提供のため、以下の第三者サービスを利用します:

  • Google AdMob — 広告配信および広告 ID 処理
    プライバシーポリシー: https://policies.google.com/privacy
  • Google Gemini API — AI 歌詞生成（非識別入力値の送信）
    プライバシーポリシー: https://policies.google.com/privacy
  • Cloudflare Workers — Gemini API 呼び出しのプロキシ（リクエスト中継）
    プライバシーポリシー: https://www.cloudflare.com/privacypolicy/

7. 保管期間

  • 端末内保存データ: 利用者が削除するまで
  • 広告 ID: Google AdMob のポリシーに従う
  • AI 歌詞生成リクエスト: 応答返却後ただちに破棄（保存なし）

8. 利用者の権利

  • データ削除: アプリ内のデータリセット機能ですべての進行記録を削除
  • 広告 ID のリセット: Android 端末の設定 > Google > 広告からいつでもリセット可能
  • パーソナライズ広告の停止: 同じ設定から無効化可能
  • アプリの削除: 端末からアプリを削除するとすべてのローカルデータが完全消去されます

9. 児童のプライバシー保護

本アプリは 13 歳以上の利用者を対象としており、13 歳未満の児童から意図的に個人情報を収集することはありません。13 歳未満の児童の情報が収集された事実を確認した場合、ただちに削除します。

10. セキュリティ

  • すべてのネットワーク通信は HTTPS (TLS) で暗号化
  • AI 歌詞生成プロキシは共有秘密鍵でクライアントを認証し不正呼び出しを防止
  • 端末内保存データは Android のアプリサンドボックス内でのみアクセス可能
  • 開発者サーバーに利用者データを永続的に保存しない

11. ポリシー変更

本ポリシーが変更される場合、アプリ内の通知または本ページの更新でお知らせします。改訂版は本ページに掲載された日から効力を生じます。
